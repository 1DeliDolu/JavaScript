Adding a Method to an Object

Adding a new method to an object is easy:

        <script>
           const person  = {
            firstName: "Deli",
            lastName: "Dolu",
            id: 4935,
          };
          person.name = function() {
            return this.firstName + " " +this.lastName;
          };
          
          document.getElementById("demo").innerHTML =
          "My father is " + person.name(); 
        </script>
        <p id="demo"></p>
