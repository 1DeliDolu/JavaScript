JavaScript Objects

JavaScript objects are mutable.

Any changes to a copy of an object will also change the original object:

JavaScript Objects are Mutable 
        Objects are mutable: They are addressed by reference, not by value.
            If person is an object, the following statement will not create
            a copy of person:

const x = person;

        The object x is not a copy of person. 
              It is person. Both x and person are the same object.

              Any changes to x will also change person, 
              because x and person are the same object.

Any changes to a copy of an object will also change the original object: 

                const person = {
                firstName:"John",
                lastName:"Doe",
                age:50, eyeColor:"blue"
              }
              
              const x = person ;
              x.age = 10 ;
