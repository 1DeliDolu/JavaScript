Using an Object Literal 
This is the easiest way to create a JavaScript Object.

        1.Using an object literal, you both define and create an object in
        one statement.
        
        2.An object literal is a list of name:value pairs (like age:50) inside curly braces {}.
        
        3.The following example creates a new JavaScript object with four properties:
        
        const person  = { firstName: "John",lastName: "Doe",age :50,eyeColor: "blue"};

        document.getElementById("demo").innerHTML =
        person.firstName  + " is " +person.age  + " years old.";

Spaces and line breaks are not important. An object definition can span multiple lines: 

       const person  = {
          firstName: "John",
          lastName: "Doe",
          age: 50,
          eyeColor: "blue"
          };
        
          document.getElementById("demo").innerHTML =
         person.firstName + " is " +person.age + " years old.";

This example creates an empty JavaScript object, and then adds 4 properties: 

        const  prson = {};
        prson.firstName = "John";
        prson.lastName = "Doe";
        prson.age =50 ;
         prson.eyeColor = "blue"; 

        document.getElementById("demo").innerHTML =
        prson.firstName + " is " +prson.age + " years old.";
