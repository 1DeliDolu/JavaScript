Nested Arrays and Objects

Nested JavaScript Objects and Arrays.

Values in an object can be another object:

Nested Objects 
        myObj = {
          name:"John",
          age:30,
          cars: {
            car1:"Ford",
            car2:"BMW",
            car3:"Fiat"
          }
        }

You can access nested objects using the dot notation or the bracket notation:

myObj.cars.car2

myObj.cars.car2; 
      const myObj = {
        name: "John",
        age: 30,
        cars: {
        car1: "Ford",
        car2: "BMW",
        car3: "Fiat"
        }
      }
      document.getElementById("dem").innerHTML = myObj.cars.car2;

or:

myObj2.cars["car2"]
