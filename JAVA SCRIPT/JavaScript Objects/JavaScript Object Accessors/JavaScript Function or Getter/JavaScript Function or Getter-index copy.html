JavaScript Function or Getter?

What is the differences between these two examples?

      Example 1 access fullName as a function: person.fullName().
      Example 2 access fullName as a property: person.fullName.
      The second example provides a simpler syntax.
JavaScript Function or Getter? Object data can be accessed using property stored as a function. 
  
        <script>
            //Create an object:
            const  person = {
            firstName : "Deli",
            lastName  : "Dolu",
            age       : 33,
            fullName  : function() {
            return this.firstName + " " + this.lastName;
            }
        };
        // Display data from the object using a method:
        document.getElementById("demo").innerHTML =person.fullName();
        </script>
        <p id="demo"></p>

JavaScript Setter (The set Keyword) Object data can be accessed using a getter. 
        <script>
            //Create an object:
            const  person = {
            firstName : "Deli",
            lastName  : "Dolu",
            age       : 33,
            get fullName()  {
            return this.firstName + " " + this.lastName;
          }
        };
        // Display data from the object using a getter:
        document.getElementById("demo").innerHTML =person.fullName;
        </script>
        <p id="demo"></p>
