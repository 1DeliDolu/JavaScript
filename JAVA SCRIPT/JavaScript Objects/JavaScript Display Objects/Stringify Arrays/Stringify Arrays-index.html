Stringify Arrays

It is also possible to stringify JavaScript arrays:

    
JSON.stringify  can stringify arrays:

  <script>
    
    const arr  = ["John", "Peter", "Sally", "Jane"];
    document.getElementById("demo").innerHTML = JSON.stringify(arr);
  </script>
  <p id="demo"></p >
