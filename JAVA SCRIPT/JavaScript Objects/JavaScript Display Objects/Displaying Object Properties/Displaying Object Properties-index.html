Displaying Object Properties

The properties of an object can be displayed as a string:

      <script>
        const person = {
          name: "Deli",
          age :  30,
          city: "New York"
        };
        
        document.getElementById("demo").innerHTML = 
        person.name + ", " +person.age + ",
        " +person.city;
      </script>
      <p id="demo"></p>
