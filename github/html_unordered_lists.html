HTML Unordered Lists

he CSS list-style-type property is used to define the style of the list item marker. It can have one of the following values:

  • disc
    • Sets the list item marker to a bullet (default)
  • circle
    • Sets the list item marker to a circle
  • square
    • The list items will not be marked
  • none
    • The list items will not be marked

Unordered List with Disc Bullets

style="list-style-type:disc;"

  • Coffee
  • Tea
  • Milk

Unordered List with Circle Bullets

style="list-style-type:circle;"

  • Coffee
  • Tea
  • Milk

Unordered List with Square Bullets

style="list-style-type:square;"

  • Coffee
  • Tea
  • Milk

Unordered List without Bullets

style="list-style-type:none;"

  • Coffee
  • Tea
  • Milk

A Nested List

Lists can be nested (list inside list):

  • Coffee
  • Tea
